Logging class, for logging things. We log to both stdout and the log file `./logs/amalgamate.log'. Logging Levels available are:

  • error
  • warn
  • info this level and above is in the log file
  • verbose
  • debug this level and above is displayed in stdout
  • silly

    Call by using logger.error(msg), logger.warn(msg), etc.
